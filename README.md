# SwiftCart

SwiftCart is an Online shop app built with the MEVN stack.

## Installation and running the project

### Installing the dependencies

```js
npm i
```

### Running the project

#### Starting the backend server

```js
npm run backend
```

#### Starting the frontend server

```js
npm run dev
```
